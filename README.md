# SpeakSecure

Privacy-first, open-source voice authentication API with OAuth 2.0 support.

This is the deployed Hugging Face Space for SpeakSecure. Documentation, architecture, and API reference are in the [`Docs/`](./Docs/) folder of this repository.

- **Demo:** open the root URL of this Space
- **API docs (Swagger):** `/docs`
- **Health check:** `/api/v1/health`
- **Status:** `/api/v1/status`

For setup, integration guide, and full project background, see [`Docs/README.md`](./Docs/README.md).
